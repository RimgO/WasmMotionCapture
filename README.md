# WCMDC

WCMDCは、Webベースの3Dシーン管理およびVRMアバターのトラッキングを行うプロジェクトです。

## 機能
- 3Dカメラ制御
- シーン管理
- VRMアバターの読み込みと管理
- トラッキング機能

## ディレクトリ構成
```
index.html
main.js
package.json
pnpm-lock.yaml
style.css
vite.config.js
src/
  camera.js
  scene-manager.js
  tracking.js
  vrm-manager.js
```

## セットアップ
1. リポジトリをクローンします。
2. 依存パッケージをインストールします。
   ```sh
   pnpm install
   ```
3. 開発サーバーを起動します。
   ```sh
   pnpm run dev
   ```
